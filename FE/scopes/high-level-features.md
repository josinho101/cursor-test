# High-Level Features for a Trello-Like Task Management Application

This document lists the high-level features needed for a task management application similar to Trello, including JWT-based authentication and collaboration capabilities.

## General Application Features

- Dark and light mode switch
- Responsive UI
- Use material UI design
- JWT based authentication and role based authorization.
- Use reactjs along with material UI components.

## 1. Authentication

- User login 
- User widget and logout 

## 2. Boards

- Create board
- Edit board name and description
- Delete board

## 3. Lists

- Create list
- Rename list
- Reorder lists by drag and drop
- Delete list

## 4. Cards and Tasks

- Create card
- Edit card title and description
- Delete card
- Move card across lists
- Reorder cards inside a list
- Assign members to cards
- Set due dates
- Add labels or tags
- Add comments on a card

## 5. Subtasks and Checklists

- Create checklist within a card
- Add checklist items
- Mark checklist items as complete or incomplete
- Reorder checklist items
- Show checklist progress percentage

## 6. Collaboration Features

- Real-time updates when other users change boards or cards
- Mention users in comments
- Notifications for assignment, due date, comments, and mentions

## 7. Search, Filter, and Sorting

- Global search across boards and cards
- Search by title, description, labels, members, and due dates
- Filter cards by member
- Filter cards by status

## 8. Admin Features

- Manage users
- Manage workspaces
